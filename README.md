# news_categorize

> a news categorization based on jieba.

- `categorize.py`: Implement a Text class to categorize news as well as get keywords from a text.

- `scel2mmseg.py`: A program which is of great help to make the words lib  expected by `categorize.py`.

- `words_lib`: Categorized words for  `categorize.py` to categorize news.

- `test_text`: Some text for test.
